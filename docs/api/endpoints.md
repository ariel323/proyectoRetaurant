# API Endpoints

## MenÃº

- `GET /api/menu` - Obtener todos los platos
- `POST /api/menu` - Crear nuevo plato
- `PUT /api/menu/{id}` - Actualizar plato
- `DELETE /api/menu/{id}` - Eliminar plato

## Mesas

- `GET /api/mesas` - Obtener todas las mesas
- `POST /api/mesas` - Crear nueva mesa
- `PUT /api/mesas/{id}` - Actualizar estado de mesa

## Pedidos

- `GET /api/pedidos` - Obtener todos los pedidos
- `POST /api/pedidos` - Crear nuevo pedido
- `PUT /api/pedidos/{id}` - Actualizar estado del pedido
