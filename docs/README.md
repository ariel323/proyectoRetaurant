# Sistema de Gestión de Restaurante

## Descripción

Sistema full-stack para gestión de restaurante con:

- Backend: Java Spring Boot
- Frontend Admin: React + TypeScript
- Frontend Cliente: React + TypeScript

## Características

- ✅ Gestión de menú
- ✅ Administración de mesas
- ✅ Sistema de pedidos
- ✅ Panel de administración
- ✅ Interfaz de cliente

## Estructura del Proyecto

```
├── backend/           # API Spring Boot
├── frontend-admin/    # Panel de administración
├── frontend-cliente/  # Interfaz de cliente
├── docs/             # Documentación
└── scripts/          # Scripts de automatización
```

## Quick Start

1. Backend: `cd backend && mvn spring-boot:run`
2. Admin: `cd frontend-admin && npm start`
3. Cliente: `cd frontend-cliente && npm start`
